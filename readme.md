# ObiJS CLI 

A CLI for obijs-cli.

## How to install

```shell
yarn global add obijs-cli
```
or 
```shell
npm -g install obijs-cli
```

## How to use

Runnig command:

```shell
$ obijs-cli new name_of_project
```
